Ez-IoT Project

Last semester I developed an application to send data to a web page and display the current status of trash containers. Using a Raspberry Pi with an ultrasonic transmitter, data is collect and send with MQTT protocol to a web server hosted with a linux distribution. Said data is processed through java script and then saved into a SQL database.
